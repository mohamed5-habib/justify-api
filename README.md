#  Text Justification API - Tictactrip Technical Test

A professional REST API for justifying text to 80 characters per line with token-based authentication and rate limiting.

##  Features

- ✅ **Token Authentication** - Secure API access via bearer tokens
- ✅ **Text Justification** - Exactly 80 characters per line
- ✅ **Rate Limiting** - 80,000 words per day per token
- ✅ **402 Payment Required** - When daily limit exceeded
- ✅ **Modern Web Interface** - Beautiful documentation UI
- ✅ **Health Monitoring** - Real-time API status
- ✅ **TypeScript** - Full type safety
- ✅ **Production Ready** - Error handling, logging, CORS
